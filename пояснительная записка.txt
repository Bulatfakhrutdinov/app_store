Проект "App Store" предназначен для скачивания приложений. При запуске web приложения пользователю необходимо авторизоваться в нём. 
Далее он попадает на главную страницу, где находятся приложения для скачивания. Если пользователь кликнет по названию приложения, то он 
попадёт на страницу этого приложения, где будет его описание и кнопка загрузки. В шапке сайта есть ссылки на страницы(загрузить приложение(открывается
форма для заполнения данных о проекте(необходимо указать название проекта, его описание, иконку проекта, zip архив с ним), другие 
пользователи(страница с именами пользователей, которых можно добавить в друзья), друзья(страница с именами ваших друзей), выйти(выход из профиля))
